土壤水文学植被分类作业
采用中国植被资料，通过ArcGIS裁剪出118.18°E~ 118.68°E，31.18°N~31.68°N范围，统计出植被分类。

操作步骤

导入中国植被资料

在工具箱中选择Conversion Tools -> 转为栅格 -> ASCII 转栅格工具，输入UMD1km China.txt文件，输出为vegetation栅格数据集，点击确定，自动生成。注意输入文件路径不宜过深，且路径中最好不要有中文，否则可能会转换失败。

设置投影坐标系

导入中国植被资料后，右下角显示未知单位，这是由于未定义投影坐标系。选择视图 -> 数据框属性 -> 坐标系 -> 投影坐标系 -> UTM -> WGS 1984 -> Northern Hemisphere -> WGS 1984 UTM Zone 50N，点击应用。在数据框属性 -> 常规 -> 单位 -> 显示中选择十进制度，点击确定。分带选择参考下图。


裁剪区域

在文档目录下新建图层clip，要素类型为面，点击确定。点击编辑器 -> 开始编辑 -> clip –> 创建要素 -> 面，在地图上点出目标裁剪区域后保存编辑。在工具箱中选择Data Management Tools -> 栅格 -> 栅格处理 -> 裁剪工具，输入vegetation图层，输出范围选择clip图层，输出栅格数据集cvege，点击确定即可裁剪出所需区域。



统计植被分类

右键cvege图层，选择属性 -> 符号系统 -> 唯一值，查看标注与计数，即为不同植被分类。

植被分类

植被分为0（water）、7（wooded grassland）、8（closed shrubland）、10（grassland）、11（cropland）、13（urban and built-up）六种。各类型占比如下表。

VALUE COUNT PROPORTION
water 67 2.4%
wooded grassland 26 0.9%
closed shrubland 12 0.4%
grassland 624 22.3%
cropland 2058 73.7%
urban and built-up 6 0.2%
